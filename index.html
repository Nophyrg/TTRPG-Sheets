Notes

These sheets have been modified by Gryphon M., and were originally created by Joe Fulgham for Adventure for Hire.

Most core systems and design choices were made by Fulgham. Gryphon has tweaked them to include some extra fields and add the additional playbooks. You can see the original website here, where Fulgham has character sheets for many other RPG's.

These sheets are (as of this writing) untested. If you notice any issues, please report them via Sheet on GitHub
